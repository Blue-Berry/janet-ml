#+title: Todo

* High Priority

** TODO [#A] with_janet / with_janet_env RAII bracket
   Create Fun.protect-based wrappers in janet_ml.ml / janet_ml.mli:
   - =with_janet : (unit -> 'a) -> 'a= — calls init, runs f, always calls deinit
   - =with_janet_env : (Janet.Table.t -> 'a) -> 'a= — creates a core_env and passes it in
   Fixes the current footgun where an exception leaks the Janet VM.

** TODO [#A] Error handling in dostring / dobytes
   Currently =dostring= silently discards the =janet_signal= return value, returning
   =Nil= on error with no indication of failure. Fix:
   - Change =dostring= to return =(janet_signal * Janet.t)=
   - Change =dobytes= likewise
   - Add =dostring_exn= / =dobytes_exn= convenience variants that raise a typed
     OCaml exception (=Janet_error of string=) on non-ok signal
   Update janet_ml.mli accordingly.

** TODO [#A] Function.call error handling
   =Function.call= calls =janet_call= which can raise a Janet panic (crashing
   the process if uncaught). Fix:
   - Make =Function.pcall= the primary surface (already exists, signal + value)
   - Add =Function.call_exn : t -> Janet.t list -> Janet.t= that wraps pcall
     and raises =Janet_error= on error signal
   - Deprecate the current silent =call=

** TODO [#A] with_root GC bracket
   Bind =janet_gcroot= / =janet_gcunroot= internally and expose only the safe
   bracket form:
   - =Janet.with_root : Janet.t -> f:(Janet.t -> 'a) -> 'a=
   This prevents use-after-GC bugs when holding Janet values across allocating
   operations (any dostring, fiber continue, etc.).

* Medium Priority

   Bind =janet_description= and =janet_pretty= in function_description.ml.
   Expose on the high-level side:
   - =Janet.to_string : Janet.t -> string=
   - =Janet.pretty : ?depth:int -> Janet.t -> string=
   Essential for debugging and the REPL demo.

** TODO [#B] Env.def / Env.var

   - =Env.def : Env.t -> string -> Janet.t -> unit=
   - =Env.var : Env.t -> string -> Janet.t -> unit=
   Currently users must run Janet source to register values, which is awkward.

** TODO [#B] Table/Struct iteration
   Bind =janet_dictionary_next= to enable iteration over Janet dicts. Expose:
   - =Table.iter : Table.t -> f:(Janet.t -> Janet.t -> unit) -> unit=
   - =Table.fold : Table.t -> init:'a -> f:('a -> Janet.t -> Janet.t -> 'a) -> 'a=
   - =Table.to_seq : Table.t -> (Janet.t * Janet.t) Seq.t=
   - =Table.of_pairs : (Janet.t * Janet.t) list -> Table.t=
   - =Struct.to_seq : Struct.t -> (Janet.t * Janet.t) Seq.t=

** TODO [#B] Fiber.cancel
   Bind =janet_cancel= in function_description.ml. Expose:
   - =Fiber.cancel : Fiber.t -> Janet.t -> unit=
   Needed for timeout / error-injection patterns.
